# Security &amp; Contact

**Lucas Green Book** is a small, free, non‑commercial project. It's a set of Python scripts run
locally to generate printable PDFs from open data — it ships **no servers, no accounts, and no
network services**, and it collects **no user data**. So the usual web/app attack surface doesn't
apply here.

## Reporting an issue

If you find a security problem, a bug, or you have a **copyright / trademark / course‑removal
request**, please email:

**📧 info@lucasgreenbook.org**

Please don't open a public issue for anything sensitive — email first and we'll respond.

## Scope

- ✅ In scope: the engine scripts in this repository, the build docs, and the `legal/` records.
- ❌ Out of scope: third‑party data sources (OpenStreetMap, USGS) — report those to their projects.

## A note on data

Every distributed book is built from **open data only** — OpenStreetMap (ODbL), public‑domain USGS
3DEP LiDAR, and published scorecard facts. No commercial product's data or imagery is used. See
[`legal/`](legal/) for the full provenance and independent‑creation record.
