# Poppy Ridge — Esri/Maxar Imagery (RESOLVED 2026-07-13)

## ✅ Resolution
The Esri/Maxar imagery has been **removed** and the Poppy Ridge aerial was **rebuilt from USDA
NAIP (U.S. public domain)**. As of 2026‑07‑13 the project contains **no Esri/Maxar, Google, or
Apple imagery anywhere** — verified. The whole project is now open‑data / public‑domain only.
- Deleted: `aerial_src.jpeg`, `GREENS_LETTERED.jpg` (both Esri‑derived).
- Rebuilt: `aerial_reference_PERSONAL.pdf` and `greenbook.pdf` page 1 now use **USDA NAIP**,
  clearly labeled **"Pre‑2025 layout — public domain — old routing, does not match current holes."**
- Kept: `GREENS_LETTERED_coords.txt` (derived factual coordinates, no imagery).

**Remaining honesty caveat (not a legal issue):** NAIP's latest flight predates the 2025
rebuild, so the aerial shows the **OLD** course. It is labeled as such and marked "site/area
reference only." A current, accurate aerial isn't possible from public‑domain sources until USGS/
NAIP re‑flies Livermore or the layout is added to OpenStreetMap.

## Background (why this was flagged)
Poppy Ridge was **rebuilt in 2025** (Jay Blasi redesign). There is **no open (OpenStreetMap)
geometry or public‑domain (USGS) elevation** of the new layout yet — verified against the
California Lidar Status service and USGS TNM (only 2006 + 2021 coverage, both pre‑rebuild), and
OpenStreetMap still shows the old routing.

Originally the Poppy aerial embedded **Esri World Imagery (Maxar)** — the only place in the
project that used a third party's copyrighted, licensed content. Esri's Master License Agreement
permits on‑screen/in‑application display and transitory caching only; it does **not** permit
exporting/redistributing the imagery or building printed/static derivatives without a separate
paid license. Baking the pixels into a PDF is a technical breach even personally, and would be
real copyright + license exposure if distributed. **That is why it was removed.**

## The hard line (still applies going forward)
- Distributed products use **open data + public domain + facts only.**
- Copyrighted/licensed imagery (Esri/Maxar, Google, Apple) never goes into anything shared.
- Merely **viewing** a copyrighted routing diagram to derive factual hole numbering is fine
  (facts aren't copyrightable); reproducing/embedding it is not.
